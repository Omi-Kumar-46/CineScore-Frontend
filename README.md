## 🎬 Overview

"Predicting Cinematic Success."
**CineScore** is an enterprise-grade Single-Page Application designed to bridge a passion for cinema with the precision of data science. Developed as a comprehensive solo capstone project, it represents the complete lifecycle of product engineering—from initial conceptualization and rigorous product management to final deployment.

Built entirely from scratch without heavy frameworks, the architecture demonstrates advanced frontend concepts including a custom window.history SPA router, dynamic data visualization, and state persistence. The interface pairs meticulous UI/UX planning with a modern design system, seamlessly blending neo-brutalism with deep glassmorphism to deliver a premium, data-driven user experience.

Our philosophy is built around **"The Director's Chair"**: bringing high-end visual fidelity (true glassmorphism, dynamic glowing spotlights, buttery-smooth SPA routing) to data-driven web applications.

## ✨ Features Highlight

- **Cinemorph UI**: A custom-engineered design language featuring true transparent glassmorphism (`rgba(0,0,0,0.3)`), hardware-accelerated animations, and responsive dark/light mode environments.
- **Buttery SPA Navigation**: Zero-reload 1000ms swipe-rendering utilizing localized State Machines to transition from the Dashboard Hub directly into the Predictive Radar.
- **Universal Spotlight Engine**: Focus interactions feature zero-wave precision edge masks (`@property` loops) that highlight critical data rows without breaking layout grids.
- **Live Hype Ticker**: A custom rotational DOM engine designed to visualize trailing buzz metrics for the next 10 theatrical releases.
- **Legacy Vault**: A responsive data-table system that stores user predictions against official box office numbers to rank and baseline analytical precision over time.

---

## 🛠️ Technology Stack

- **Core**: HTML5, Vanilla JavaScript (ES6+), Vanilla CSS3.
- **Architecture**: Custom lightweight Single-Page Application (SPA) routing, modular `DOMContentLoaded` feature injection.
- **Design Rendering**: Advanced CSS `@property` interpolations, sub-pixel mask composites, and layered `backdrop-filter` blurs for maximum performance without standard framework bloat.
- **Database (Mock)**: Local Storage APIs utilized to simulate live user sessions, data persistence, and the Auth Controller.

## 🌐 Deploying & Hosting

To experience CineScore natively:

1. Clone or download this repository.
2. Ensure you have the following **critical files** pushed to your Github Pages / Vercel deployment:
   - `index.html` (The Public Landing Splash)
   - `Prediction.html` (The Core Prediction SPA engine)
   - `Hub.html` (The Command Center & Watchlist Dashboard)
   - `script.js` (The Central Brain)
   - `Base.css` (The Global stylesheet & Cinemorph tokens)
   - `/Assets/` directory (All posters, backgrounds, and brand logos)
3. The platform is entirely responsive and fully functional as a static site. No Node.js initialization required for the frontend shell.
